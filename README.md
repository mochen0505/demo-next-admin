# README #

demo-next-admin

### Installation ###

* Install dependencies

    ```Bash
    npm install
    ```

### Run the App ###
 
* For all users, simply run:

    ```Bash
    npm run dev
    ```
    
* The App will be running on localhost:3002

### Deployment ###

* To build, run:

  ```Bash
  next build
  next start
  ```
